## file-upload-multer

Upload an excel file to a post route and store it in the file system

###### Install dependencies
```JavaScript
npm install
```
###### Run tests
```JavaScript
npm test
```
###### Start Project
```JavaScript
npm start
```


```JavaScript
POST /api/upoad
```

###### Payload
```JavaScript
excelFile: 'sample.xls'
```
###### Headers
```JavaScript
'Content-Type': 'multipart/form-data'
```
